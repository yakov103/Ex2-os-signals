## Run Locally

Clone the project

```bash
  git clone https://github.com/yakov103/Ex2-os-signals.git
```

Go to the project directory

```bash
  cd Ex2-os-signals
```

compile and run the program

```bash
  make
```

requried to have ["make","gcc","git"] installed
